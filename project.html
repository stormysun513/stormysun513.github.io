ParGMRES - A Parallel Linear Solver

The goal of project is to explore parallel techniques we learned in the class. We implemented our own general minimal residual algorithm and parallelized it using both OpenMP and CUDA. GMRES is an iterative method to solve linear system Ax = b. Due to its iterative fashion, the algorithm is inherently sequential. What we focus is to parallelize those matrix operations.

Semi-supervised Learning in Image Classification

This project is to improve performance of image classification using unlabeled image data. The dataset is cifar-10 with 5000 labeled images, 45000 unlabeled images, and 10000 images for testing. I tried two methods to solve this problem. The first one is to use Kmeans with spatial pyramid matching while the second one use CNN with self-training approach.

Web Application - CookByYourself

A website that collects recipes from different websites and estimates prices required to make those dishes. Our web server is developed using Django framework with PostgreSQL and deployed on the AWS EC2 machine. Users are able to register an account, comments, and search recipes. Data are collected via Selenium and python BeautifulSoup library.

Real-Time Resource Reservation and Enforcement Framework

We built an energy-aware CPU resource reservation and enforcement framework in Linux kernel. Users can specify timing constraints of a task and register it as a real-time task in our framework. The system will then schedule these tasks based on these constraints while still running other normal tasks maintained by Linux OS at the same time. We also made a android application able to log framework configuration and status of these tasks using android NDK.

Deep Learning in Speech Recognition

The project is to build a speech recognition system predicting English sentences from speech data using deep learning techniques. A speech recognition system comprises of two parts: acoustic feature transformation and the language model construction. We used a deep neural network and structured SVM for feature extraction and used LSTM-RNN to rescore the possible output generated by the N-best Viterbi inference algorithm.

2013

Functionally Reduced And-Inverter Gates Simulator

A circuit optimization tool that reads a circuit representation file and abstracts it as an acyclic graph. We customized the memory allocation of user-defined class by overriding the new and delete operation in C++. Graph traversal approaches like BFS and DFS are used to find unreachable gates and propagate simulation results. Explicit free lists are used to manage available memories.
