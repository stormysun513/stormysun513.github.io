Research Projects

BioScope

Advisor: Prof. Hao-Hua Chu

The goal of this project is to create an extensible bandage system with sensory and processing components that can be stacked like LEGO blocks. Using this smart bandage system, nurses can conveniently customise and configure these sensor blocks to remotely monitor physiological data of patients and assess their physical condition in hospitals.

My contribution:

  • Designed the mockup of outer casing through software 123D Design and prototyped with 3D printers.
  • Implemented pedometers and position detection with accelerometers, heartbeat counter with ADC in PIC32 embedded system.
  • Integrate a gesture detection modules with SSD1306 OLED display as an user interface for the bandage system.
  • Bridged the data transfer between PIC32 and mobile phone with TI CC254x Bluetooth module.

Rehab diary

Advisor: Prof. Hao-Hua Chu

This project entails a phone support application to help Ketamine-dependent users to self-monitor their drug use and motivate them to maintain abstinence. A Ketamine-dependent user is requested to perform a saliva-screening test by using the screening device in the view of his or her smartphone camera. The screening result is then sent to the smartphone through Bluetooth and then forwarded to a backend server in which his or her psychiatrist can remotely monitor the patient data.

My contribution:

  • Implemented driver with OV2640 embedded camera with STM32 Discovery board.
  • Came up with a circuit to detect whether the test strip inside the screening device is soaked.
  • Designed the inner structure with lens and camera to accommodate the entire test strip but also reduce the size of device.
  • Implemented the android interface with Bluetooth Low Energy API to retrieve images from screening device.

Course Projects

Offline speech recognition system (Machine Learning and Having It Deep)

Instructor: Prof. Hung-Yi Lee

This project is aimed to build an offline speech recognition system with some existing toolkits or algorithms implemented in a few assignments in the class. First, we were given raw speech data, which can be transform into both MFCC or fBank features. The recognition system is comprised of two parts: the acoustic model and the language model. We utilised Deep Neural Network combined with HMM to get the predicted state sequence. The phonemes can be generated through WFST and then forward to Recurrent Neural Network or NN with long short-term memory nodes.

My contribution:

  • Surveyed Deep Neural Network (DNN) toolkits such as libdnn and Keras.
  • Preprocessed raw acoustic data to feature vectors.
  • Implemented Viterbi algorithm with state likelihood outputted by DNN and probabilities of state transition from training data.

Functionally Reduced And-Inverter Graph Simulator (Data Structure and Programming)

Instructor: Prof. Chung-Yang Huang

It was the final project of Data Structure and Programming. I implemented a command line based software to parse an And-Inverter Graph and represents it as a data structure in the software. Several commands were provided to optimise the circuit structure and reducing the functionally equivalent candidate subgraphs. For example, unused gate sweeping, trivial optimization, structural hash and functional equivalent pairs.
